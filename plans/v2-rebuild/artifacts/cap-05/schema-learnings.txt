
# schema learnings
                                 Table "public.learnings"
       Column        |           Type           | Collation | Nullable |      Default
---------------------+--------------------------+-----------+----------+-------------------
 id                  | uuid                     |           | not null | gen_random_uuid()
 attached_to_kind    | text                     |           | not null |
 attached_to_id      | uuid                     |           | not null |
 title               | text                     |           | not null |
 statement           | text                     |           | not null |
 context             | text                     |           |          |
 created_by_actor_id | uuid                     |           | not null |
 created_at          | timestamp with time zone |           | not null | now()
 deactivated_at      | timestamp with time zone |           |          |
Indexes:
    "learnings_pkey" PRIMARY KEY, btree (id)
    "idx_learnings_actor" btree (created_by_actor_id, created_at)
    "idx_learnings_attached" btree (attached_to_kind, attached_to_id, created_at)
Check constraints:
